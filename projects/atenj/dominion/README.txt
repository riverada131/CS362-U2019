Instructions to use makefile:

1. type make and press enter to compile and run all tests to generate the report called "unittestresults.out".

2. type "make runalltests" and press enter to compile and run all tests to generate the report called 
"unittestresults.out" and "cardtestresults.out".

3. type "make unittestresults.out" and press enter to compile and run all tests to generate the 
report called "unittestresults.out".

4. type "make cardtestresults.out" and press enter to compile and run all tests to generate the 
report called "cardtestresults.out".

5. type "make clean" and press enter to remove the unittestresults.out cardtestresults.out files.

6. type "make cleanall" and press enter to remove the unittestresults.out unittest1 unittest3 
unittest2 unittest4 unittest5 *.gcda *.gcno *.gcov files.

7. type "make cleanallcard" and press enter to remove the cardtestresults.out cardtest1 
cardtest2 cardtest3 cardtest4 cardtest5 *.gcda *.gcno *.gcov files.

8. type "make cleanalltests" and press enter to remove the unittestresults.out unittest1 
unittest3 unittest2 unittest4 unittest5 cardtestresults.out cardtest1 cardtest2 cardtest3 
cardtest4 cardtest5 *.gcda *.gcno *.gcov files.
